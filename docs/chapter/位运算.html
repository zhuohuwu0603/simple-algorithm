1. 位运算

1.1. 六种基本运算

  1. 与 a & b
  2. 或 a | b
  3. 取反 ~a
  4. 异或 a ^ b
  5. 左移 <<
  6. 右移 >>

1.2. 技巧型运算

1.2.1. x & -x

lowbits运算: lowbit(x) = x & -x

应用: 得到x的二进制表示中最右边的一个1

推导: 负数 = 正数的补码 + 1

1.2.2. 模拟加法

a ^ b相当于 a 与 b 的二进制的不进位相加

配合移位操作可以实现加法操作

1.2.3. 模拟乘以2的幂

x << k 相当于 x * 2的k次幂

1.2.4. 模拟除以2的幂

x >> k 相当于 x / 2的k次幂

比如：我们在使用二分算法时，获取mid，可以 使用 mid = l + (r - l) >> 1

1.2.5. 模拟整除判断

(x & m) == 0 相当于 x % (m + 1) == 0

我们常用 x & 1 == 0 来判断 x是否是偶数就是利用此条性质

举例：(x & 3) == 0 相当于 x % 4 == 0

1.2.6. 异或运算交换两个元素

a = a ^ b
b = a ^ b
a = a ^ b

1.2.7. 将第k位置为1(k从0开始)

x |= (1 << k)

1.2.8. 将第k位置为0

x &= ~(1 << k)

1.2.9. 判断第k位是不是1

(x >> i) & 1 或者 x & (1 << i)

1.2.10. 删除最后一位的1

  • x & (x - 1)
  • x ^ (x & -x)
