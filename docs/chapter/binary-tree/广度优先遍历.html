1. 广度优先遍历

也叫 层遍历，BFS

遍历过程是从上到下，一层一层的，更接近人的思维

1.1. 实现方式

BFS 需要 借助 队列 来实现

1.2. 应用场景

  • 求解最短、最小路径问题
  • 图的拓扑排序
  • 最短路问题求解（dijkstra算法等）

1.3. 优点和缺点

优点：

  • 接近人的思维，易于理解
  • 不会爆栈
  • 求最短路

缺点：

  • 代码实现较长
